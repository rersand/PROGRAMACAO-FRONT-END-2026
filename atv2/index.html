guia de estudo para linguagem C\C++

1. Oque é a linguagem C ?

A linguagem C foi criada por Dennis Ritchie nos laboratórios da Bell Telephone em 1972.

C foi criada com um propósito: ser usada no desenvolvimento de uma nova versão do sistema operacional Unix. A

primeira versão do Unix utilizava Assembly. Então podemos dizer que desde o principio C foi uma linguagem

criada por programadores para programadores.(Eduardo Casavella, O que é Linguagem C?)




2. Estrutura

A estrutura de um programa em C é composta por diretivas de pré-processador

(#include), declaração de variáveis globais, protótipos de funções e a função

principal int main(), onde a execução inicia. Todo bloco de código é delimitado por

chaves {} e instruções terminam com ponto e vírgula ;.




3. Utilizações

A linguagem C é amplamente utilizada para desenvolver software de baixo nível,

sistemas operacionais (como Linux e Windows), drivers de hardware, sistemas

embarcados (cafeteiras, painéis de carros), bancos de dados (MySQL, Oracle) e

motores de jogos, devido à sua alta eficiência, velocidade e controle direto sobre os

recursos do sistema, como memória e processador.
